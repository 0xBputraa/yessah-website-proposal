# Available Options

## Option 1 — Free Setup

The website can be deployed immediately using the server IP address.

### Benefits

- No additional costs
- Fully functional
- Ready to use

### Limitations

- Accessed via IP address instead of a custom domain
- Less professional for branding and marketing purposes

### Cost

| Item | Cost |
|--------|--------|
| Monthly Cost | $20 |
| Domain | $8/year |

---

## Option 2 — Custom Domain (Recommended)

Deploy the website using a custom domain (e.g. `yessahblessah.com`).

### Benefits

- Professional branding
- Easier for users to remember
- HTTPS enabled
- Better user trust and credibility

### Additional Feature: Tweet Verification

Users must publish a predefined tweet before generating an invite code.

#### Workflow

1. User clicks **Generate Code**
2. User publishes the required tweet
3. System verifies the tweet
4. Invite code is generated

This helps increase organic exposure and community growth.

### Cost

| Item | Cost |
|--------|--------|
| Domain | $10–15/year |
| Hosting | Included |
| SSL | Included |

---

## Option 3 — X OAuth Verification

Advanced verification using X (Twitter) authentication.

### Workflow

1. User logs in with X
2. System verifies required follows
3. Invite code is generated only after verification

### Benefits

- Harder to bypass
- Higher quality engagement
- Accurate follower verification

### Requirements

- X API Basic Plan

### Cost

| Item | Cost |
|--------|--------|
| X API Basic | $100/month |
| Domain | $10–15/year |
